# Reviews

This folder is archival.

Use it for past code reviews, audits, and one-off inspection notes.

Do not use files here as the source of truth for current teleport direction.

Current active teleport planning lives in:

- `super_turtle/docs/REPO_BOUND_TELEPORT_SPEC.md`
- `super_turtle/docs/TELEGRAM_WEBHOOK_POC.md`

Guidelines:

- keep new review files narrowly scoped and dated
- prefer naming like `review-<topic>.md`
- avoid turning this folder into a backlog or product-plan dump
- move stale planning notes into the active spec docs or delete them instead of layering more review files on top
