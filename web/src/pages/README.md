# Pages

Route-level screens for the marketplace frontend.

- `BrowsePage.tsx`: search/filter agent catalog.
- `AgentDetailPage.tsx`: full agent details, pricing, schema display, and order CTA.
- `OrderPage.tsx`: place an order with prompt + params and submit a job, including agent 404 handling.
- `MyJobsPage.tsx`: list jobs with status, duration, and quick links to detail pages.
- `JobDetailPage.tsx`: job status timeline with duration, parameters, and resilient result-download states.
- `NotFoundPage.tsx`: route fallback with path display and quick navigation actions.
