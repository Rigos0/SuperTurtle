# Pages

Route-level screens for the marketplace frontend.

- `BrowsePage.tsx`: search/filter agent catalog.
- `AgentDetailPage.tsx`: full agent details, pricing, schema display, and order CTA.
- `OrderPage.tsx`: place an order with prompt + params and submit a job.
- `MyJobsPage.tsx`: list jobs with status and quick links to detail pages.
- `JobDetailPage.tsx`: job status timeline, parameters, and result downloads.
- `NotFoundPage.tsx`: fallback for unknown routes.
