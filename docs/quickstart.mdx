## Bot Commands

Once the bot is running, these commands are available:

| Command | Purpose |
|---------|---------|
| `/start` | Show status and confirm your user ID |
| `/new` | Start a fresh conversation |
| `/resume` | Pick from recent sessions to continue where you left off |
| `/context` | Show how much of your context quota you've used |
| `/stop` | Stop the current task (or just type `stop`) |
| `/status` | Check what Claude is doing right now |
| `/looplogs` | Show the last 50 lines of bot logs |
| `/restart` | Restart the bot |

## Troubleshooting

**Bot doesn't respond to messages**
- Verify your Telegram user ID is correct: send `/start` and check the ID shown
- Confirm the bot token was set correctly in `.env`
- Restart the bot: Ctrl+C in the terminal, then `bun run start` again
- Check logs: `tail -f /tmp/claude-telegram-bot-ts.log`

**Claude authentication fails**
- For Claude Code: run `claude` in a terminal and verify you're logged in
- For Codex: run `codex` and verify you're logged in
- Check that your subscription is active

**Voice messages fail**
- Ensure `OPENAI_API_KEY` is set in `.env`
- Verify the key is valid at [OpenAI console](https://platform.openai.com/)

**Setup wizard got stuck**
- Press Ctrl+C to exit and try again
- Verify Bun is installed: `bun --version`
- Check that Claude/Codex CLI is in your PATH: `which claude` or `which codex`
