# Quickstart

Set up Super Turtle and start coding from Telegram. The setup is AI-guided: you provide credentials and the system handles install and configuration.

## Prerequisites

Before you start:

- `bun` installed ([bun.sh](https://bun.sh/))
- Claude Code or Codex CLI installed and authenticated
- A Telegram bot token from [@BotFather](https://t.me/BotFather)
- Your Telegram user ID (used for `TELEGRAM_ALLOWED_USERS`)
- Optional `OPENAI_API_KEY` if you want voice transcription

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Rigos0/superturtle.git
    cd superturtle
    ```
  </Step>

  <Step title="Open Claude Code or Codex at repo root">
    ```bash
    claude

    # or
    codex
    ```
  </Step>

  <Step title="Say the setup prompt">
    In Claude Code or Codex, send:
    ```
    Set up Super Turtle on this machine.
    ```
  </Step>

  <Step title="Provide required values">
    Share the values requested by the setup wizard:
    - `TELEGRAM_BOT_TOKEN`
    - `TELEGRAM_ALLOWED_USERS`
    - Optional `OPENAI_API_KEY`
  </Step>

  <Step title="Start the Telegram bot">
    ```bash
    cd super_turtle/claude-telegram-bot
    bun run start
    ```
  </Step>

  <Step title="Message your bot on Telegram">
    Open Telegram, find your bot, and send:
    ```
    /start
    ```
  </Step>
</Steps>

<Info>
The setup flow writes `super_turtle/claude-telegram-bot/.env`, installs dependencies, and sets the default driver automatically.
</Info>

## BotFather Setup

Use [@BotFather](https://t.me/BotFather) to create and configure your bot:

1. Run `/newbot` and finish the prompts.
2. Save the generated token as `TELEGRAM_BOT_TOKEN`.
3. Run `/setcommands` and paste:

```text
start - Show status and user ID
new - Start a fresh session
resume - Pick from recent sessions
context - Show context usage
stop - Interrupt current query
status - Check what Claude is doing
looplogs - Show bot logs
restart - Restart the bot
```

## Getting Your Telegram User ID

Use one of these:

- Message [@userinfobot](https://t.me/userinfobot) and copy your numeric ID.
- Or run `/start` in your bot chat and read the ID in the response.

Set that value as `TELEGRAM_ALLOWED_USERS`.
